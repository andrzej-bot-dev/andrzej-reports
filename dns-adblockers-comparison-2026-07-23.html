⚡ TL;DR — Co wybrać zamiast NextDNS?

  1. Chcesz za darmo i bez limitów? → AdGuard DNS Public (94.140.14.14) — nielimitowany, zero rejestracji, działa z Android Private DNS
  2. Chcesz customizację jak NextDNS ale za darmo? → AdGuard Home (self-hosted na Raspberry Pi) — pełny dashboard, per-device, encrypted DNS built-in
  3. Chcesz najlepszą prywatność? → Mullvad DNS (adblock.dns.mullvad.net) — Szwecja, zero logów, anycast, nielimitowany
  4. Chcesz app na Androida z firewallem? → RethinkDNS — 190+ blocklist, firewall per-app, open source
  5. NextDNS za 25 SEK/mies jest OK — jeśli potrzebujesz per-device profiles i analytics, to nadal najlepsza opcja paid
⚠️ Ważne: DNS blokuje ~60-70% reklam ogółem. Nie zablokuje YouTube ads, sponsored postów na FB/IG, ani Google Search ads. Do pełni szczęścia potrzebujesz DNS + uBlock Origin w przeglądarce.

📋 Blocklisty — Które wybrać?

HaGeZi (⭐ rekomendowane 2026)

  • Light: ~297K domen — minimalny, nie psuje nic
  • Normal: ~450K domen — codzienna ochrona, false positive-free
  • Pro: ~500K domen — najlepsza ochrona (⭐ sweet spot)
  • Ultimate: ~3M domen — agresywny, może coś zepsuć
  • GitHub: github.com/hagezi/dns-blocklists

Inne godne polecenia:

  • Steven Black Unified Hosts: ~220K, konserwatywny, zintegrowany z Pi-hole
  • 1Hosts (Pro): ~400K domen — rywal HaGeZi
  • OISD Big: ~430K — zero false positives (celowo)
  • Developer Dan (Ads + Tracking): 42K + 12K — minimalne ale skuteczne
  • DNS Blocklists (UK): 320K — zintegrowane z dnsmasq
📐 Zasada: HaGeZi Pro jako baza + OISD jako uzupełnienie = najlepszy stosunek ochrony do false positives. Nie przesadzaj z rozmiarem — większa lista ≠ lepsza ochrona.

🔧 Zaawansowane Setup'y

Tailscale + Pi-hole/AdGuard Home (mobilna ochrona)

Zabierz swój DNS ze sobą — Pi-hole/AGH w domu + Tailscale VPN na telefonie.

  • Tailscale free: do 100 urządzeń
  • Override DNS w tailnet → Twój Pi/AGH
  • Android: Tailscale app + exit node = pełna ochrona poza domem
  • WireGuard alternatywa: tańszy, lżejszy, ale mniej features
✅ Ochrona wszędzie, nie tylko w domu
✅ Pełna kontrola, zero cloud dependency
❌ Wymaga Tailscale/WireGuard na telefonie
❌ Latencja = odległość do domu (nie problem w Sztokholmie)

AdGuard Home + Upstream Encrypted DNS

Najlepszy combo self-hosted: AGH jako lokalny DNS + Mullvad/Quad9 jako upstream.

Upstream DNS servers:
https://dns.mullvad.net/dns-query (DoH)
tls://adblock.dns.mullvad.net (DoT)
https://dns.quad9.net/dns-query (DoH fallback)

Bootstrap: 9.9.9.9, 194.242.2.3
✅ Encrypted upstream + lokalny cache = szybki i prywatny
✅ Pełna customizacja blocklist
